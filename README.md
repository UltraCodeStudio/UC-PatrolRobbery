# UC-PatrolRobbery


## FiveM Robbery System with patrolling guards

- Highly customisable via config.lua
  - Locations are easily changeable/removable

- Features several configuration options
  - Adding new locations
  - Adding/Changing guard patrols and guard detection distance
  - Adding lootables and setting the props and search animations
  - super easy to add your own skill checks in the config

- Customisable points for Selling materials
  - Nicely animated for better immersion

- Supports QBOX
- Easily adaptable to your own framework
